# SGJong

MP Mahjong with SG Rules

## Requirements

- Yarn

## Available Commands

| Command      | Description                                                                     |
| ------------ | ------------------------------------------------------------------------------- |
| `yarn`       | Install project dependencies                                                    |
| `yarn dev`   | Build project and open web server running project                               |
| `yarn build` | Builds code bundle with production settings (minification, uglification, etc..) |
